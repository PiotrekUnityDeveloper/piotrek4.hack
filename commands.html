commands list:

.bind <hack> <key> - binds a module
.toggle <hack> - toggles a specified module
.masstoggle - toggles all modules
.help - helps u
.panic - disables all modules
.bind clearall - removes all hackbinds
.launch <height> - launches you in the air
.teleport <x> <y> <z> - locally teleports the player to the specified coordinates
.website - opens Piotrek4Hack website!
